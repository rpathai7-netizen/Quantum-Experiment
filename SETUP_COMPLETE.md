## 🔧 Troubleshooting

Problem: "ModuleNotFoundError: No module named 'qiskit'"
- Solution: `pip install qiskit qiskit-aer numpy`

Problem: "Insufficient memory" error
- Solution: Platform auto-switches to analytical methods for large circuits

Problem: Unicode/encoding errors (Windows)
- Solution: Already fixed! System uses ASCII-safe characters

Problem: Cloud connection errors
- Solution: Install provider: `pip install qiskit-ibm-runtime`
- Get API key from respective cloud platform
