### Troubleshooting:
1. **GUI won't launch?**
   - Check: `python --version` (need 3.8+)
   - Try: `pip install -r requirements.txt`
   - Run: `python gui_application.py`

2. **EXE won't run?**
   - Need: Windows 10/11, 500 MB space
   - Try: Run as Administrator
   - Download: Fresh copy from releases

3. **Import errors?**
   - Run: `pip install -r requirements.txt`
   - Verify: `pip list | grep qiskit`
   - Reinstall: `pip install --upgrade -r requirements.txt`

4. **Network issues?**
   - Local simulation: Works offline
   - Cloud backends: Need internet
   - Check: Firewall settings
